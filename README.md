# bank-marketing-predictor

基于银行电话营销数据的交互式数据分析与在线认购预测系统。

## 功能

- **数据分析看板**：多维度可视化探索营销数据
- **在线预测系统**：输入客户特征，实时预测是否会认购定期存款

## 技术栈

Python 3.11 · Streamlit · scikit-learn · pytest · ruff · Docker

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt -r requirements-dev.txt

# 启动应用
streamlit run app.py --server.port 8004
```
